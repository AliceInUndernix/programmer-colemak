| <backspace\> | <escape\>    |
| g            | b            |
| [ ]          | <backspace\> |
| <caps lock\> | <control\>   |
| d            | g            |
| h            | k            |
| z            | x            |
| x            | c            |
| c            | d            |
| b            | <circle\>    |
| k            | m            |
| m            | h            |
| <fn\>        | z            |
| <meta\>      | <fn\>        |
